## HASHTAAG

### Prerequiest

 1. Node.js 12x
 2. Mongodb
 
### Project Setup
 1. `https://github.com/ankit030592/hashtaag.git`
 2. `cd hashtaag`
 3. `npm install`
 4. Project runs on port 3000

### Routes
 1. http://localhost:3000/state?q=stateName
 2. http://localhost:3000/town?q=townName
 3. http://localhost:3000/district?q=districtName
